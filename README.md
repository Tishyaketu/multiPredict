# Multi-Predict

Multi-Predict is a comprehensive multi-disease prediction platform designed to predict heart disease, diabetes, breast cancer, and lung cancer.

## Features
- User Authentication
- Heart Disease Prediction (Logistic Regression)
- Diabetes Prediction (SVM)
- Breast Cancer Prediction (CNN)
- Lung Cancer Prediction (InceptionResNet)
- Prescription Upload Support
- PDF Report Generation

## Tech Stack
- MongoDB, Express, React, Node.js (MERN)
- Python (scikit-learn, tensorflow)
